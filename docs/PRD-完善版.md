# 衣橱管理与穿搭建议 APP 完善版 PRD

## 1. 产品定位

- 产品名称：`ClosetMate / 衣橱管家`
- 核心价值：帮助用户高效管理衣物，并在不同天气、场景和风格偏好下快速获得可执行穿搭方案
- 目标用户：20-34 岁城市女性为主，兼顾男性与学生群体
- 核心卖点：
  - 让衣服被“看得见、找得到、搭得出”
  - 降低每日穿搭决策成本
  - 减少重复购买与闲置浪费

## 2. 首版上线范围

首版按 MVP 交付，覆盖 P0 主流程：

- 首页天气 + 穿搭推荐
- 衣柜录入、编辑、删除
- 搜索、筛选、常用标记
- 自定义搭配与穿搭保存
- 个人资料与偏好设置
- 本地持久化
- 可选 Supabase 图片存储

以下能力列入后续版本：

- AI 吊牌扫描
- 自动图像识别分类
- 登录注册与跨端同步
- 穿搭社区分享
- 通知提醒与运营后台

## 3. 用户主流程

1. 首次进入 App
2. 设置昵称、城市、风格偏好
3. 添加 5-10 件核心衣物
4. 首页选择场景并查看推荐
5. 保存喜欢的推荐穿搭
6. 自定义调整并形成常用搭配库

## 4. 页面结构

- 首页：天气、场景切换、智能推荐、统计概览
- 衣柜：搜索、分类筛选、单品卡片、详情页、添加衣物
- 穿搭：查看已保存穿搭、分享、标记今日穿搭
- 我的：个人资料、风格偏好、恢复演示数据
- 二级页：衣物表单、衣物详情、自定义搭配

## 6. 推荐逻辑

首版采用规则引擎，不依赖大模型即可上线：

- 场景匹配优先
- 季节和温度匹配优先
- 常用与偏好风格加分
- 待清洗、损坏、隐藏衣物降权或排除
- 中性色和低色彩冲突组合优先

## 7. 成功指标

- 7 日留存
- 人均录入衣物数
- 首页推荐点击率
- 推荐保存率
- 自定义搭配创建率
- 闲置衣物占比变化

## 8. 上线验收标准

- 主流程无阻塞性崩溃
- 衣物 CRUD 全流程可用
- 推荐逻辑可稳定返回 3-5 套方案
- 本地数据关闭重开后仍保留
- Android 测试包可安装
- iOS TestFlight 包可分发

## 9. 技术方案

- 前端：Expo React Native
- 状态管理：Zustand
- 本地存储：AsyncStorage
- 云端可选：Supabase
- 天气：Open-Meteo
- 打包：EAS Build
